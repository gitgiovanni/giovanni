Development SISTEMA DE RASTREIO

Interface integrada para monitoramento de pedido e gestão de status logístico em tempo real. Para esse projeto eu desenvolvi ele por completo, fiz o front end focado em UX e navegação rápida, buildei o banco de dados em mysql dentro de um windows server e desenvolvi duas API´s, uma para conexão do banco de dados com o site, e outra para conexão do banco de dados com o Protheus (O ERP da empresa, que guarda os pedidos e seus status). Site em plena operação.

Process improvement TOTVS FLUIG

Desenvolvimento completo utilizando o serviço Fluig, desde a home que funciona como o intranet e principal meio de comunicação entre os colaboradores, ao desenvolvimento de processos estruturados utilizados diáriamente por todos os colaboradores, como o formulário dechamado de TI, o formulário de geração de Link de pagamento e o formulário de envio de contas a pagar para o administrativo.
